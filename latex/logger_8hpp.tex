\doxysubsection{Detailed Description}
Logging Utilities for detecting and selecting logging interfaces. 

Defines the high-\/level wrapper for logging data on the Data Gator (DG). Currently this consists of a single function {\ttfamily log\+\_\+data} which takes the data strings to be logged and logs them to all available logging destinations such as MQTT and the SD card.

Data destinations are controlled by whether their \textquotesingle{}is available\textquotesingle{} flag is set to {\ttfamily true}. This flag is first set in the user configuration files but can be overridden if the interface cannot be initialized during setup.
