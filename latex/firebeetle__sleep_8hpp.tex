\doxysubsection{Detailed Description}
This file implements firebeetle sleep (ESP32) 

Implements wrapper functions used to put the uC processor of the ESP32 into a power conserving sleep mode. The deep\+\_\+sleep function configures the ESP32 to turn off RTC peripherals but still wake from sleep without external prompting via the reset pin.
